# 🚀 GrowthOS - Web3 Productivity Dashboard

A clean, modern Web3-style personal productivity dashboard for tracking daily discipline, habits, trading, content creation, and growth activities.

## ✨ Features

### 📊 Daily Dashboard
- **Trading Tracker**: Max 1-2 trades with RR ratio, result, and reasoning
- **Career/Work**: Job search, DMs sent counter, opportunities explored
- **Content Creation**: Instagram Reels, YouTube, Twitter tracking
- **Health Metrics**: Gym, walks, water intake
- **Learning/Building**: Study and project development tracking

### 🔥 Gamification
- **Streak System**: Daily completion streak with fire animation
- **Weekly Progress**: Visual M-T-W-T-F-S-S progress bar
- **Confetti Celebration**: Animated reward on task completion
- **Progress Ring**: Real-time circular progress indicator

### 📝 Content Strategy
- **30-Day Planner**: Pre-built content calendar with Hook/Topic/CTA structure
- **Editable Entries**: Click any text to customize your strategy
- **Trending Ideas**: AI-suggested viral Web3 topics
- **Completion Tracking**: Mark days as complete

### 📚 History & Analytics
- **Activity Journal**: Browse all previous days
- **Edit Entries**: Modify past records (hover to see edit button)
- **Delete Records**: Remove unwanted entries
- **Filter Options**: View by trading days or completed days only

### 🤖 AI Suggestions
- **Productivity Tips**: Daily actionable advice
- **Content Ideas**: Fresh Web3 content inspiration
- **Growth Strategy**: Networking, skills, and authority building tips

## 🎨 Design
- **Dark Theme**: Solana-inspired purple/blue/cyan gradients
- **Glassmorphism**: Modern frosted glass UI elements
- **Responsive**: Mobile-friendly layout
- **Smooth Animations**: CSS transitions and confetti effects
